Elysium Scapes

Advanced UI • Prototyping • Component Systems • Figma

El Reto: Diseñar la interfaz y la capa de micro-interacciones para una plataforma web orientada al sector del lujo (Elysium Scapes). El objetivo principal era transmitir exclusividad puramente a través del comportamiento dinámico de la web, utilizando técnicas avanzadas de prototipado.

1. User Flow & Dirección de Arte

Mapeo exhaustivo del viaje del usuario (User Journey) para una plataforma de reservas premium. Definición de una dirección de arte sofisticada (Dark Mode) con tipografía serif elegante, priorizando que la fotografía de los destinos (Santorini, Bali) guíe la experiencia de navegación.

2. Smart Animate & Componentes Interactivos

Construcción de un sistema modular de componentes en Figma utilizando variantes complejas. Configuración de transiciones de alta fidelidad mediante Smart Animate (curvas bezier personalizadas y efectos de muelle/spring) para previsualizar con precisión matemática las físicas e interacciones antes de pasar a desarrollo.

Bakeneko Bakery

E-Commerce Premium • Product Strategy • Frontend (GSAP/JS) • Phywebpal

El Reto: El reto era tan delicado como los propios Wagashi: diseñar una experiencia de e-commerce que no se sintiera como una compra, sino como la entrada a una ceremonia. Quería fusionar tres mundos que me fascinan; las criaturas del folclore japonés (Yokai), la repostería artesanal (Wagashi) y el ritual del maridaje con cafés y vinos de especialidad. El objetivo es crear el espacio digital de una marca de lujo silencioso, pensada para el antojo ocasional, no para la cesta diaria.

1. Diseño de Experiencia

El primer acercamiento con usuarios reveló un riesgo: la estructura clásica de tienda online empujaba al producto hacia una falsa sensación de ‘compra diaria’, algo que chocaba con la esencia del proyecto. La solución no fue cambiar el producto, sino el contexto. Para preservar ese aura de consumo ocasional y especial, rediseñé cada punto de contacto visual.

2. Rompiendo el E-commerce Tradicional

Diseñé la experiencia con enfoque Mobile-First y una cesta de compra asíncrona oculta en modales: el usuario nunca abandona la narrativa visual para comprar, eliminando fricciones. A nivel de sistema, la web está pensada como un ecosistema multi-idioma (ES, EN, JP). Esto va más allá de traducir textos: el logotipo muta sus variantes tipográficas según el idioma activo, manteniendo la coherencia estética en cada versión.

3. Ilustración y Gamificación como Hilo Narrativo

El factor diferencial no está solo en el código, sino en el universo visual que construí alrededor del producto. Cada Wagashi se asocia a un Yokai protector, representado mediante una ilustración original. Estas ilustraciones no son decorativas: forman parte de un sistema de cartas coleccionables físicas y digitales. El sistema de fidelización se cierra con un incentivo final: reunir las 6 variedades da acceso a un postre único, convirtiendo la repetición de compra en una pequeña búsqueda mitológica personal.

4. El Oráculo: Asistencia sin fricción

Como parte del ecosistema phygital, diseñé un componente narrativo para los momentos de duda. El Oráculo es un botón situado estratégicamente durante la selección de producto; si el usuario no se decide entre los Wagashi, el Oráculo elige uno de forma aleatoria, manteniendo la coherencia con el tono enigmático de la marca. No es una herramienta fría de filtrado, sino un gesto casi lúdico que resuelve la indecisión sin romper la atmósfera de la experiencia.

Brand Assets • E-Commerce Design • Custom Graphics • Client Work

1. CraftLove Journals (E-Commerce)
